# Event Photos Directory

Place event photos here to display in the volunteer calendar.

The `events.json` file references these images:
- `colleyville-lions.jpg` - Colleyville Lions volunteer event
- `ncl-volunteers.jpg` - NCL volunteer day
- `saturday-session.jpg` - Saturday open sessions
- `corporate-team.jpg` - Corporate team building events
- `school-group.jpg` - School volunteer groups

Photos will automatically fall back to a placeholder if the image file is not found.

## Image Requirements
- Format: JPG, PNG, or WebP
- Recommended size: 800x600px or 4:3 aspect ratio
- Keep file sizes under 500KB
- Optimize before uploading (use TinyPNG or similar)
